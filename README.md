# Proyecto Tienda

## Instalación
1. Clonar el repositorio
2. Ejecutar `npm install` para instalar dependencias
3. Ejecutar `npm run dev` para iniciar el proyecto en modo desarrollo

## Uso
- Login de admin: usuario `admin`, contraseña `admin123`
- Login de usuario: usuario `user`, contraseña `user123`
- Agregar productos al carrito, eliminar o vaciar carrito
- Admin puede agregar/editar/eliminar productos
- Buscar productos y navegar con paginación

## Tecnologías
- React.js + Context API
- TailwindCSS
- React Router
- React Icons
- React Toastify
- MockAP
